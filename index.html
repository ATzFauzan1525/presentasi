üí∞ Tampilan di UI:

‚ö° Kenapa Otomatis?
  • reduce() dipanggil setiap kali component render
  • Saat cart berubah ‚Üí React re-render ‚Üí reduce() jalan lagi
  • Tidak perlu tombol "Hitung Total" atau refresh
  • Real-time reactive = Core konsep React!
